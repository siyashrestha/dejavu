How it works

Learn Easy is a free language-learning tutorial, offering language course to English and Nepali speakers.The program is completely free to use, with no advertisement or hidden fees. After the language from which user desires to learn has been selected, user gets the option of starting right from the beginning (Basics 1), which presumes user has no prior background in the language. User also have the option of taking a shortcut and directly jumping to later lessons after performing some tests/giving an exam. Placement test in the web version, to see what level user might best fit in.

Each lesson is composed of two types of exercises: a translation exercise where users are required to translate from native language (L1) to target language (L2) or vice-versa. Each lesson is composed of about 10 exercises. It becomes harder as progressed.
